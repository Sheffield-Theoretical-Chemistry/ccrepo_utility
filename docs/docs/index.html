ccRepo: A python utility for accessing the Hill Group’s ccRepo basis sets

Note

This project is under active development.

About

ccRepo is an open-source database for correlation-consistent basis sets. It is designed to retrieve correlation consistent basis sets from the Hill Group ccRepo. Please note that basis sets can also be downloaded directly from the web based platform as well as by using this python library.

Acknowledging usage

If you use this information in your research, please consider citing the ccRepo website as part of your paper.

You should also cite the original literature detailing the basis sets you use. A full bibliography can be found on ccRepo to help with this.

In addition it would be greatly appreciated if you would cite the ccRepo utility GitHub repository if you used this package in your work. :)
