ccRepo: A python utility for accessing the Hill Group's ccRepo basis sets
==================================

About
======

ccRepo is an open-source database for correlation-consistent basis sets. It is designed to retrieve correlation consistent basis sets from the `Hill Group ccRepo <http://www.grant-hill.group.shef.ac.uk/ccrepo/>`. Please note that basis sets can also be downloaded directly from the web based platform as well as by using this python library.
